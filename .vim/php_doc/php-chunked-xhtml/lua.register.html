Lua::registerCallback — PHP 関数を Lua に登録

説明

public void Lua::registerCallback ( string $name , callback $function )

PHP 関数を Lua に "$name" という関数として登録します。

パラメータ

name

function

有効な PHP 関数コールバック

返り値

例

例1 Lua::registerCallback()の例

<?php
    $lua = new Lua();
    $lua->registerCallback("echo", "var_dump");
    $lua->eval(<<<CODE
    echo({1, 2, 3});
CODE
);
?>

上の例の出力は、 たとえば以下のようになります。

array(3) {
  [1]=>
  float(1)
  [2]=>
  float(2)
  [3]=>
  float(3)
}
