Imagick::getImageMatte

(PECL imagick 2.0.0)

Imagick::getImageMatte — 画像がマットチャネルを持っているかどうかを返す

説明

bool Imagick::getImageMatte ( void )

画像がマットチャネルを持っている場合に TRUE、 そうでない場合に false を返します。 このメソッドは、ImageMagick バージョン 6.2.9 以降で Imagick をコンパイルした場合に使用可能です。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
