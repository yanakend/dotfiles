Imagick::previousImage

(PECL imagick 2.0.0)

Imagick::previousImage — オブジェクト内の前の画像に移動する

説明

bool Imagick::previousImage ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

画像リスト内の前の画像を Imagick オブジェクトに関連付けます。

返り値

成功した場合に TRUE を返します。
