HaruEncoder::getUnicode

(PECL haru >= 0.0.1)

HaruEncoder::getUnicode — 指定した文字を unicode に変換する

説明

int HaruEncoder::getUnicode ( int $character )

指定した文字を unicode に変換します。

パラメータ

character

変換する文字のコード。

返り値

エラー / 例外

エラー時に HaruException をスローします。
