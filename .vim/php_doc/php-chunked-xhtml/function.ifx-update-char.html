ifx_update_char

(PHP 4, PHP <=5.2.0)

ifx_update_char — 文字オブジェクトの内容を更新する

説明

bool ifx_update_char ( int $bid , string $content )

指定した文字オブジェクト bid に関して 文字オブジェクトの内容を更新します。

パラメータ

bid

文字オブジェクト ID。

content

新規データの文字列。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
