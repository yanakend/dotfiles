要件

This extension requires » libeio library. The following installs libeio: 

$ touch ~/.cvspass
$ cvs -z3 -d :pserver:anonymous@cvs.schmorp.de/schmorpforge co libeio
$ cd libeio
$ ./autogen.sh
$ ./configure
$ make
# make install
Note, the last command needs root priveleges.
