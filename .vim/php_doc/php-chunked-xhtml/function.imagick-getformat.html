Imagick::getFormat

(PECL imagick 2.0.0)

Imagick::getFormat — Imagick オブジェクトのフォーマットを取得する

説明

string Imagick::getFormat ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

Imagick オブジェクトのフォーマットを返します。

返り値

画像フォーマットを返します。 エラー時に ImagickException をスローします。
