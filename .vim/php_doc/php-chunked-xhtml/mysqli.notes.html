ノート

若干の実装ノート

  1. MYSQL_TYPE_GEOMETRY に対するサポートが MySQLi エクステンションに PHP 5.3 で追加されました。

  2. MYSQL_TYPE_GEOMETRY 型の項目を操作するために libmysql および mysqlnd の中にそれぞれ別の異なる内部実装があることに注意してください。 一般的に言って、mysqlndでのメモリ割り当ては かなり少なくなります。 例えば、POINT 項目が結果セットにあると、 メモリ上で POINT 項目を保持するには 50 バイトも 必要ないのに、libmysql は最高 4GB の RAM を事前に割り当てるかもしれません。 mysqlnd を使うと、 メモリの割り当てはずっと小さく、50 バイト未満です。
