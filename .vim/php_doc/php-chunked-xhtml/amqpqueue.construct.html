AMQPQueue::__construct

(PECL amqp >= Unknown)

AMQPQueue::__construct — AMQPQueue オブジェクトのインスタンスを作成する

説明

AMQPQueue::__construct ( AMQPConnection $amqp_connection [, string $queue_name = "" ] )

ブローカ上の AMQP キューを表す AMQPQueue のインスタンスを作成します。

パラメータ

amqp_connection

有効な AMQPConnection オブジェクト。

queue_name

作りたいキューの名前。名前を省略した場合は、ブローカが名前を生成します。

返り値
