ImagickDraw::render

(PECL imagick 2.0.0)

ImagickDraw::render — これまでのすべての描画コマンドを画像上にレンダリングする

説明

bool ImagickDraw::render ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

これまでのすべての描画コマンドを画像上にレンダリングします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
