Imagick::setGravity

(バージョン情報なし。おそらく SVN 版にしか存在しないでしょう)

Imagick::setGravity — gravity を設定する

説明

bool Imagick::setGravity ( int $gravity )

Imagick オブジェクトのグローバル gravity プロパティを設定します。 このメソッドは、ImageMagick バージョン 6.4.0 以降で Imagick をコンパイルした場合に使用可能です。

パラメータ

gravity

gravity プロパティ。 gravity 定数 の一覧を参照ください。

返り値

値を返しません。
