AMQPExchange::bind

(PECL amqp >= Unknown)

AMQPExchange::bind — バインドする

説明

public bool AMQPExchange::bind ( string $queue_name , string $routing_key )

指定したルーティングキーを使って、exchange をキューにバインドします。

パラメータ

queue_name

バインドするキューの名前。

routing_key

バインド時に使うルーティングキー。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

エラー / 例外

失敗した場合に AMQPExchangeException をスローします。
