Imagick::mosaicImages

(PECL imagick 2.0.0)

Imagick::mosaicImages — 画像からモザイクを作成する

説明

Imagick Imagick::mosaicImages ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

画像シーケンスをはめ込み合成して単一の画像にします。 シーケンス内の各画像を、画像のページオフセットで指定した位置に合成したものを返します。

返り値

成功した場合に TRUE を返します。
