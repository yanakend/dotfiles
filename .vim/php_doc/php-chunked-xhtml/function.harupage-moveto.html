HaruPage::moveTo

(PECL haru >= 0.0.1)

HaruPage::moveTo — 新しい描画パスの開始位置を設定する

説明

bool HaruPage::moveTo ( float $x , float $y )

新しい描画パスの開始位置を定義します。

パラメータ

x

新しい開始位置の座標。

y

新しい開始位置の座標。

返り値

成功した場合に TRUE を返します。

エラー / 例外

エラー時に HaruException をスローします。
