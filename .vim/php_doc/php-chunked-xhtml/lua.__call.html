Lua::__call

(PECL lua >=0.9.0)

Lua::__call — Lua 関数を呼び出すための Lua 用マジックメソッド

説明

public void Lua::call ( string $lua_func , array $args )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

lua_func

lua での関数名

args

Lua 関数に渡される引数

返り値

例

例1 Lua::call()の例

<?php
    $lua = new Lua("script.lua");
    $lua->func_name("foo", "bar")); /** func_name は、script.lua で定義された関数 */
?>
