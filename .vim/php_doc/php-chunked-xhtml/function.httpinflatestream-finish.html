HttpInflateStream::finish

(PECL pecl_http >= 0.21.0)

HttpInflateStream::finish — 展開ストリームを終了する

説明

public string HttpInflateStream::finish ([ string $data ] )

展開ストリームを終了します。展開ストリームは、 終了した後でも再開することが可能です。

パラメータ

data

展開するデータ。

返り値

展開されたデータの最後の部分を返します。
