PDF_set_info

(PHP 4 >= 4.0.1, PECL pdflib >= 1.0.0)

PDF_set_info — ドキュメント情報のフィールドを設定する

説明

bool PDF_set_info ( resource $p , string $key , string $value )

ドキュメント情報フィールド key に value を設定します。成功した場合に TRUE を、失敗した場合に FALSE を返します。
