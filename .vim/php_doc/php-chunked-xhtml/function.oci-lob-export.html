OCI-Lob->export

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob->export — LOB の内容をファイルに出力する

説明

bool OCI-Lob::export ( string $filename [, int $start [, int $length ]] )

LOB の内容をファイルに出力します。

パラメータ

filename

ファイルへのパス。

start

出力を開始する位置。

length

出力するデータの長さ。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
