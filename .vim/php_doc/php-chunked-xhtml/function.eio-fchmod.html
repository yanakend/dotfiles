eio_fchmod

(PECL eio >= 0.0.1dev)

eio_fchmod — Change file permissions.

説明

resource eio_fchmod ( int $fd , int $mode [, int $pri = EIO_PRI_DEFAULT [, mixed $callback = NULL [, mixed $data = NULL ]]] )

eio_fchmod() changes permissions for the file specified by fd file descriptor.

パラメータ

fd

The file descriptor, e.g. returned by eio_open().

mode

The new permissions. E.g. 0644.

pri

リクエストの優先順位。EIO_PRI_DEFAULT、EIO_PRI_MIN、EIO_PRI_MAX あるいは NULL を指定します。 NULL を渡した場合、pri は内部的に EIO_PRI_DEFAULT となります。

callback

callback 関数が、リクエスト終了時にコールされます。 この関数のプロトタイプは、

void callback($data, $result);
に一致しなければなりません。 $data は、リクエストに渡すカスタムデータです。 $result にはリクエスト固有の結果が格納されます。基本的には、対応するシステムコールが返すそのままの値となります。

data

Arbitrary variable passed to callback.

返り値

eio_fchmod() returns request resource on success or FALSE on error.
