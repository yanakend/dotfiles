Imagick::minifyImage

(PECL imagick 2.0.0)

Imagick::minifyImage — 画像をその半分のサイズに比例縮小する

説明

bool Imagick::minifyImage ( void )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

画像のサイズを元の半分に比例縮小します。

返り値

成功した場合に TRUE を返します。
