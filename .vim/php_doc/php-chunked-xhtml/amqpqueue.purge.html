AMQPQueue::purge

(PECL amqp >= Unknown)

AMQPQueue::purge — キューの中身をパージする

説明

public bool AMQPQueue::purge ( string $queue_name )

キューの中身をパージします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

queue_name

パージするキューの名前。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
