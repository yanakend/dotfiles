Imagick::getOption

(PECL imagick 2.0.0)

Imagick::getOption — 指定したキーに対応する値を返す

説明

string Imagick::getOption ( string $key )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

オブジェクト内の指定したキーに対応する値を返します。

パラメータ

key

オプションの名前。

返り値

指定したキーに対応する値を返します。 エラー時に ImagickException をスローします。
