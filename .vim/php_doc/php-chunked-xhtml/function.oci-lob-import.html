OCI-Lob->import

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Lob->import — ファイルデータを LOB にインポートする

説明

bool OCI-Lob::import ( string $filename )

filename からのデータをラージオブジェクトの現在の位置に書き込みます。

パラメータ

filename

ファイルへのパス。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
