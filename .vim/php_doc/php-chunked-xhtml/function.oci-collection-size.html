OCI-Collection->size

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Collection->size — コレクションのサイズを返す

説明

int OCI-Collection::size ( void )

コレクションのサイズを返します。

返り値

コレクション内の要素の数、もしくはエラー時に FALSE を返します。
