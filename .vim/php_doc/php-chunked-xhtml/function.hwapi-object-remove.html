hw_api_object->remove

(PHP 4, PHP 5 < 5.2.0, PECL hwapi SVN)

hw_api_object->remove — 属性を削除する

説明

bool hw_api_object::remove ( string $name )

指定した名前の属性を削除します。

パラメータ

name

属性の名前。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
