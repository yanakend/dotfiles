MongoGridFS::put

(PECL mongo >=1.0.8)

MongoGridFS::put — ファイルをデータベースに格納する

説明

public mixed MongoGridFS::put ( string $filename [, array $extra = array() ] )

パラメータ

filename

ファイルの名前。

extra

保存するファイルに追加したいその他のメタデータ。

返り値

保存したオブジェクトの _id を返します。
