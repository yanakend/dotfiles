バージョン 説明
5.3.0 MySQL エクステンションをすべてサポートする、MySQL ネイティブ・ドライバが追加されました。 (例えば mysql、mysqli および PDO_MYSQL) mysqlnd で適切な構成スイッチに渡すことにより、このサポートが有効になります。
5.4.0 MySQL ネイティブ・ドライバが MySQL エクステンションのすべてで現在デフォルトです。 (例えば mysql、mysqli および PDO_MYSQL) mysqlnd で構成に渡すことは、現在任意です。

MySQL データベース・エクステンションは、 MySQL クライアント・ライブラリを用いて構成されなければいけません。 MySQL ネイティブ・ドライバを使用するためには、MySQL データベース・エクステンションを MySQL ネイティブ・ドライバのサポート付でコンパイルすることを指定して PHP をビルドする必要があります。 これは、PHP のソースコードをビルドする前に、構成オプションを通じて行なわれます。

例えば、MySQL ネイティブ・ドライバを用いて MySQL エクステンション、mysqli と PDO MYSQL をビルドするには、以下のコマンドが渡されます。 

./configure --with-mysql=mysqlnd \
--with-mysqli=mysqlnd \
--with-pdo-mysql=mysqlnd \
[other options]

Windows へのインストール

5.3 以降の公式な PHP Windows ディストリビューションでは、 MySQL ネイティブ・ドライバは、デフォルトで使用可能です。 そのため、使用するための追加の構成は不要です。 この場合は MySQLデータベース・エクステンションは全て MySQL ネイティブ・ドライバを使用します。
