Imagick::queryFonts

(PECL imagick 2.0.0)

Imagick::queryFonts — 設定したフォントを返す

説明

array Imagick::queryFonts ([ string $pattern = "*" ] )
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

設定したフォントを返します。

パラメータ

pattern

クエリパターン。

返り値

フォントを含む配列を返します。 エラー時に ImagickException をスローします。
