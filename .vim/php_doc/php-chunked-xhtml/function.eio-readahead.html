eio_readahead

(PECL eio >= 0.0.1dev)

eio_readahead — Perform file readahead into page cache

説明

resource eio_readahead ( int $fd , int $offset , int $length [, int $pri = EIO_PRI_DEFAULT [, mixed $callback = NULL [, mixed $data = NULL ]]] )

eio_readahead() populates the page cache with data from a file so that subsequent reads from that file will not block on disk I/O. See READAHEAD(2) man page for details.

パラメータ

fd

The file descriptor

offset

Starting point from which data is to be read.

length

Number of bytes to be read.

pri

リクエストの優先順位。EIO_PRI_DEFAULT、EIO_PRI_MIN、EIO_PRI_MAX あるいは NULL を指定します。 NULL を渡した場合、pri は内部的に EIO_PRI_DEFAULT となります。

callback

callback 関数が、リクエスト終了時にコールされます。 この関数のプロトタイプは、

void callback($data, $result);
に一致しなければなりません。 $data は、リクエストに渡すカスタムデータです。 $result にはリクエスト固有の結果が格納されます。基本的には、対応するシステムコールが返すそのままの値となります。

data

Arbitrary variable passed to callback.

返り値

eio_readahead() returns request resource on success or FALSE on error.
