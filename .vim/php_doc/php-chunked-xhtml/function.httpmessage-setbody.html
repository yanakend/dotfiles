HttpMessage::setBody

(PECL pecl_http >= 0.14.0)

HttpMessage::setBody — メッセージの本文を設定する

説明

public void HttpMessage::setBody ( string $body )

HttpMessage の本文を設定します。

注意:

それに応じてヘッダを適切に更新することを忘れないようにしましょう。

パラメータ

body

新しいメッセージ本文。
