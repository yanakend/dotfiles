Lua::call

(PECL lua >=0.9.0)

Lua::call — Lua 関数を呼び出す

説明

public void Lua::call ( mixed $lua_func , array $args )

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

lua_func

lua での関数名

args

Lua 関数に渡される引数

返り値

例

例1 Lua::call()の例

<?php
$lua = new Lua();
$lua->eval(<<<CODE
  function dummy(foo, bar)
    print(foo, ",", bar)
  end
CODE
);
$lua->call("dummy", array("Lua", "geiliable\n"));
var_dump($lua->call(array("table", "concat"), array(array(1=>1, 2=>2, 3=>3), "-")));
?>

上の例の出力は、 たとえば以下のようになります。

Lua,geiliable
string(5) "1-2-3"
